# Task: Definir Secoes do Site

## Objetivo
Projetar a estrutura de secoes e o user flow completo do site.

## Entregaveis
1. Lista de secoes com hierarquia e proposito
2. Wireframe descritivo de cada secao (layout textual)
3. Fluxo de navegacao entre secoes
4. Paleta de design tokens (cores, espacamentos, tipografia)

## Secoes Sugeridas
- **Hero** — Titulo principal + animacao 3D (particle morphing atual)
- **About** — Quem e o Lucas, o que faz
- **Portfolio/Work** — Projetos e cases
- **Skills/Tech** — Stack e competencias (oportunidade pra 3D interativo)
- **Contact** — Formulario ou links de contato
- **Footer** — Redes sociais, copyright

## Criterios de Aceitacao
- [ ] Cada secao tem proposito claro definido
- [ ] Hierarquia visual documentada (o que chama mais atencao)
- [ ] Navegacao mobile definida (menu hamburger? scroll?)
- [ ] Design tokens exportados em CSS Custom Properties
